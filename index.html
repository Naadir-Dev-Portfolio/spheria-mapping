◎ The big picture read me first

Spheria is two things that now act as one:

  • Local Spheria — lives on E:\Spheria. Its brain is a local model (Ollama, currently qwen3-coder:30b) running on your own GPU. No cloud. It has ~43 real tools (files, shell, Python, browser, email, cloud drives, music…). It runs a small web server on XXX.XXX.XXX.XXX:8470 that everything talks to.
  • The Python widget (ESO / "Spheria 2.0") — the floating desktop HUD on C:\…\SPHERIA 2.0. Its brain is Gemini (cloud). It oversees your ESO agents (intel brief, food, schedule, health). It is also a door to the local brain via a ☁/🦁 toggle.

So you have a local brain (private, free, offline) and a cloud brain (Gemini, smarter for big reasoning), and they can each borrow the other's hands. The local brain can phone Gemini (ask_gemini) and run ESO agents (eso_agent); the widget can route your typing to either brain.

⭐ The mental model: every way you reach Spheria (terminal, GUI, phone, right-click, the widget) is a "door". All doors lead to the same server → the same agent loop → the same brain + tools. Learn that one spine and everything else is just a different doorway onto it.
